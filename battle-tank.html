BATTLE TANK

Battle Tank is an open world head-to-head tank fight game with AI controlled enemies and advanced control system created in Unreal Engine 4.

Will you be able to defeat the enemies? In order to do so you will have to wisely use the strategical spots distributed around the map.

Platform: PC (WINDOWS)
Team size: 1 (myself)
Project length: 4 months


DOWNLOAD (WINDOWS)

OVERVIEW

Battle Tank is a project in which I had the opportunity to put in practice all the learning that I got from taking part several times in the Weekly Game Jam using Unreal Engine 4, and from finishing the UE4 C++ Developer Course created by the GameDev team in collaboration with Epic Games.

During the development of this game, I learned the basics for the creation of 3D action games such as physics simulation, trajectory prediction, and input control. Moreover, I could expand my C++ programming learning some of the best architectures for game creation.

My goal with this project was to create a game with an advanced input control system with a fast reaction to the requests of the players and a challenging AI system.

WHAT I DID

- Coded the control system of the player and the enemy tanks.

- Coded the AI system of the enemies (Pathfinding + FSM).

- Coded the physics and the collision management.

- Created a Low Poly Shader for the battleground.

- Modeled some of the 3D assets.

- Created some of the textures.
