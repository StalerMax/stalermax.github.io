PROJECT STARFALL

Project Starfall is a 2D arcade endless game where you have to dodge the different obstacles, such as asteroids or enemy lasers, by moving the main character around the map.

Platform: Browser
Team size: 1 (myself)
Project length: 2 months


PLAY IN BROWSER

OVERVIEW

This project started as a challenge to myself by making a simple and funny video game from scratch in my spare time, and explore really interesting concepts such as the game feel or the juice.

The original idea of Project Starfall was inspired by the LUDUM DARE games of Brakeys (LUDUM DARE 38) and Blackthornprod (LUDUM DARE 40).

I'm developing Project Starfall because I want the people to feel like they are playing an old arcade game but different thanks to my personal touch. Because of those reasons I'm drawing the Project Starfall assets with a pixel art style, and I'm trying to add my personal touch using a glow filter that makes the game feel really different and impresive.

I am really happy with the result that I am getting, but I still have a hard work to do!

WHAT I DID

- Created all the art assets.

- Created all the sound effects (but the music).

- Scripted the whole game.

- Created all the game scenes.

- Created a glow shader.

- Designed the UI and gameplay.
